# 🚀 Tecnologias Usadas
* React Native + Expo
* NativeBase
* TypeScript
* FireBase / FireStore / Auth
* Phosphor-Icons'

<br> 

# 🔧 Components

* Usestate
* UseEffect
* React Navigation
* dateFormat

<br> 

# 📚 Funcionalidades
* Abrir solicitação de servicos.
* Aplicar observações.
* Registrar data de abertura das solicitacões.
* Registrar data de fechamento das solicitacões.
* Encerrar solicitacões.
